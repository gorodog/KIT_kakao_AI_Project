<외국어성적우수장학금 ‘제출점수’ 입력방법>
<공통절차>
<‘원스톱서비스 로그인’→ ‘학생지원’→ ‘교내장학금’→‘외국어성적우수 제출점수 입력 ’  → ‘외국어성적 기준점수 입력내역’에서 기준점수(환산점수) 확인 → (아래)‘A’또는 ‘B’ >

<A ><“외국어성적 제출점수 입력 대상점수 목록”에서 선택할 제출점수가 있을 경우>
<제출점수 선택 → ‘제출점수 저장’ → ‘외국어성적 기준점수 입력내역’에서 제출점수(환산점수) 확인 → 신청 완료 >
<B ><“외국어성적 제출점수 입력 대상점수 목록”에서 선택할 제출점수가 없을 경우>
<‘제출점수 추가’클릭 → 공인외국어시험 입력화면으로 이동 → ‘공인외국어시험성적표 목록’에 ‘조회’ → ‘추가’ → 시험성적 등 입력 → 저장 → 해당자료 체크박스 체크 → ‘평가자료등재신청 바로가기’→ 조회 → 체크박스 체크 → 평가자료 세부내역 확인 → 파일버튼 클릭하여 증빙자료파일 업로드 → 승인요청 →  학과 승인 → [이후 A방법과 동일]   >
