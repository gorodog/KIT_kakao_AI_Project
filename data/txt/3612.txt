<외국어성적우수장학금 ‘기준점수’ 입력 방법>
<공통절차>
<‘원스톱서비스 로그인’→ ‘학생지원’→ ‘교내장학금’→‘외국어성적우수 기준점수 입력 ’  → (뒷페이지)‘A’또는 ‘B’ >

<A ><“외국어성적 기준점수 입력 대상점수 목록”에서 선택할 기준점수가 있을 경우>
<‘외국어성적 기준점수 입력내역’에서 ‘이전학기 수혜내역’의 기존제출점수를 확인(‘조회’ 클릭) → 선택1) 이전학기 수혜내역의 수혜년도/학기가 2020학년도 1학기인 경우는 ‘기존제출점수를 현재기준점수로 저장’클릭 → ‘외국어성적 기준점수 입력내역’에서 기준점수(환산점수) 확인 → 완료  → 선택2) 이전학기 수혜내역의 수혜년도/학기가 2020학년도 1학기가 아닌 경우는‘외국어성적 기준점수 입력 대상점수 목록’에서 기준점수를 선택하여 저장 → ‘외국어성적 기준점수 입력내역’에서 기준점수(환산점수) 확인 → 완료 >
<B ><“외국어성적 기준점수 입력 대상점수 목록”에서 선택할 기준점수가 없을 경우>
<‘기준점수 추가’클릭 → 공인외국어시험 입력화면으로 이동→‘공인외국어시험성적표 목록’에서 ‘조회’→‘추가’클릭 → 시험성적 등 입력 → 저장 → 해당자료 체크박스 체크 → ‘평가자료등재신청 바로가기’→ 조회 → 체크박스 체크 → 평가자료 세부내역 확인 → 파일버튼 클릭하여 증빙자료파일 업로드 → 승인요청 →  학과 승인 → [이후 A방법 선택2와 동일]   >
